📲 Pasang Nur.ai di HP
Nur.ai adalah PWA (Progressive Web App) — bisa dipakai tanpa App Store, langsung dari browser!
1
Buka di Chrome / Samsung Internet
Pastikan kamu membuka file ini di browser Chrome atau Samsung Internet di HP Android.
2
Tap menu ⋮ (titik tiga) di pojok kanan atas
Lalu pilih "Add to Home screen" atau "Tambahkan ke Layar Utama".
3
Ketuk "Add" / "Tambahkan"
Ikon Nur.ai akan muncul di home screen. Buka seperti app biasa, fullscreen tanpa browser bar!
4
Selesai! Buka dari home screen
Nur.ai siap dipakai offline untuk surah-surah populer. Tetap butuh internet untuk audio qori.
💡 Tips Android: Jika tidak muncul opsi install, pastikan URL dimulai dengan https:// dan coba di Chrome versi terbaru.
1
Buka di Safari (wajib Safari)
PWA di iPhone hanya bisa dipasang via Safari. Tidak bisa dari Chrome atau Firefox di iOS.
2
Tap tombol Share 🔗 di bawah layar
Ikon kotak dengan panah ke atas, di bagian bawah atau atas browser Safari.
3
Scroll ke bawah → "Add to Home Screen"
Pilih "Add to Home Screen" atau "Tambahkan ke Layar Utama".
4
Ketuk "Add" di pojok kanan atas
Ikon Nur.ai muncul di home screen iPhone. Buka → fullscreen seperti native app!
💡 Tips iPhone: iOS 16.4+ mendukung notifikasi dari PWA. Update iOS untuk pengalaman terbaik.
1
Buka di Chrome atau Edge di laptop/PC
Nur.ai bisa dipasang sebagai desktop app di Windows, Mac, dan Linux.
2
Klik ikon install di address bar
Chrome: ikon komputer/download di ujung kanan address bar. Edge: ikon serupa. Atau lewat menu ⋮ → "Install Nur.ai".
3
Klik "Install" di popup konfirmasi
NurAI akan terbuka sebagai window terpisah tanpa browser bar, seperti app desktop.
💡 Tips PC: NurAI tampak di Start Menu (Windows) atau Applications (Mac). Bisa dibuka via keyboard shortcut.
